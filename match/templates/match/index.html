Grasp

We specialize in study buddying

About Grasp

This website was created for the class, CS 3240 at the University of Virginia. The purpose of the application is to find specific study partners with certain skills and majors. Users can create their own profile, list their skills and majors, and find others whom they would want to work with. The application's unique features include the ability to search for students with specific skills and majors, and a feature to save profiles they like. If you have any questions for us, visit the contact page. Or, if you have any suggestions, please help us by visiting the Suggest page.

Grasp's Features

Advanced Search Features

Grasp allows user to search for potential study buddies in an advanced and efficient way. Find study buddies based on name, major, or skill.

Look on: Homepage

Joint Calendar System

Grasp allows users to notify others of group events on a joint calendar. Simply type in the name of the event, a description, the date, and time.

Look on: Calendar

Built in Messaging

Grasp allows users to message others in a convenient manner. Users can compose a new message to anybody, view incoming messages, and view messages they have sent.

Look on: Message Center
